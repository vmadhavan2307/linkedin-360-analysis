## 🎓 Learning Git (Optional)

If you want to use Git locally instead of uploading through GitHub:

See `GITHUB-SETUP-GUIDE.md` for complete instructions.

Quick version:
```bash
# Create folder
mkdir linkedin-360-analysis
cd linkedin-360-analysis

# Initialize git
git init

# Copy all files into folder

# Stage files
git add .

# Commit
git commit -m "Initial commit: LinkedIn 360 skill"

# Create repo on GitHub, then push
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/linkedin-360-analysis.git
git push -u origin main
```
